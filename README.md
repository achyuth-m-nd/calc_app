# Python Calculator

A simple Python calculator project — built for practising Git and GitHub workflows.

## Files

| File | Description |
|------|-------------|
| `main.py` | Entry point — runs sample calculations |
| `calculator.py` | `Calculator` class with add, subtract, multiply, divide |
| `utils.py` | Helper functions (greet, get_history) |
| `test_calculator.py` | Unit tests using `unittest` |

## How to Run

```bash
# Run the calculator demo
python main.py

# Run the tests
python -m unittest test_calculator.py
```

## Concepts Practiced

- Classes and methods
- Error handling (division by zero)
- Unit testing
- Modular code structure
